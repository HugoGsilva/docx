# Pasta de Templates

Esta pasta deve conter os arquivos de template originais:

## Arquivos Necessários

1. **modeloB.odt** - Template RPBANK
   - Placeholders: `{{DATA}}`, `{{PROPOSTA}}`, `{{TOTAL}}`, `{{VALORLIQUIDO}}`, `{{VALORPROPOSTA}}`, `{{TAXAINTERMEDIACAO}}`, `{{INTERMEDIACAO}}`, `{{PARCERIA}}`, `{{ESCRITORIO}}`

2. **Modelo para proposta.docx** - Template SD-RESOLV
   - Placeholders: `{{NOME}}`, `{{NRPROCESSO}}`, `{{VALOR_LIQUIDO}}`

## Instruções

Coloque seus arquivos de template nesta pasta antes de executar a aplicação.

Os placeholders devem estar exatamente no formato `{{NOME_DO_CAMPO}}` dentro dos documentos.
